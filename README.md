# Device Finder

Rocket Team !

DLUT生产实习项目 in Rockwell Automation

 通过UDP等协议对当前子网或远程子网中的主机进行访问

基于Qt开发gui界面

src文件夹为Qt工程文件以及源码，可在Qt6环境下编译运行

release文件夹下  exe文件可以直接运行
